# Cursor Landing Page Clone

A recreation of the [Cursor](https://cursor.com/) homepage using raw HTML and CSS. This project focuses on replicating the layout, visual styling, and responsiveness of the original design without using JavaScript frameworks.

**Live Demo:** https://preetmax85.github.io/landing-page-1/

## 📝 Sections Recreated

I have successfully cloned the following sections end-to-end:

1.  **Navigation Bar** 
2.  **Hero Section** 
3.  **Logo Cloud** 
4.  **Feature Highlights** 
5.  **Testimonials**
6.  **Frontier Section** 
7.  **Changelog** 
8.  **Recent Highlights**
9.  **Final CTA**
10. **Footer** 

## 🎨 Design System

### Fonts
* **Primary Font:** `Inter`
    * Weights used: 400, 450, 500, 600, 700
* **Source:** Google Fonts

### Color Palette
The design uses a "Warm Dark" theme.

* **Backgrounds:**
    * Main Body: `#14120b` (Deep Warm Dark)
    * Cards/Panels: `#1b1913`, `#1E1C14`
    * Highlights/Hover: `#27251d`
* **Text:**
    * Headings: `#EDECEC` (Off-White)
    * Subtext/Muted: `#9a9996`, `#7A7A7A` (Warm Gray)
* **Accents:**
    * Primary Action: `#EDECEC` (White buttons)
    * Links/Highlights: `#FF4F00` (Cursor Orange)
    * Borders: `#2A2820`
